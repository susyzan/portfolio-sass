Arrakis - Spice Emporium

This e-commerce web application is one of the main academic projects and is probably the most important one.
The project began with gathering the user requirements and involved all aspects of a web appli cation development: User Requirements from Use Cases, Design Specification, Front End, Databas e Design and Development and has an object oriented approach.
All the documents produced are available upon request.

The website is built on a custom MVC PHP framework and uses a MySql database. The front end interface uses Bootstrap.

This site allows for different account types, including the Admin account and dashboard.
Feel free to register and try out all the functionalities.
To login as an administrator ,please contact me in order to receive the credentials.
